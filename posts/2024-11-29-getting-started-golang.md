# Getting Started with Go

Go is a statically typed, compiled programming language designed at Google.

## Key Features

1. Simple and clean syntax
2. Built-in concurrency support
3. Strong standard library

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```
